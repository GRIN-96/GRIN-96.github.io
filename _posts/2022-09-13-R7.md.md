# 리스트

- 리스트 - 리스트 생성
    - 백터와 달리 값이 서로 다른 데이터 타입 담을 수 있다.
        - “이름” 키에 “홍길동
        - “학과” 키에 “컴퓨터 공학과”
        - “학번” 키에 20188115
    - list ( key1 = value1, key2 = value2, ··· )

- 행렬 - 행렬 생성
    - matrix(데이터, 행의수(nrow),  열의 수(ncol), byrow=FALSE, dimnames=NULL)
        - byrow를 TRUE로 설정하면 행우선, FALSE 일 경우 열 우선
        - **DEFAULT** 값 FALSE
