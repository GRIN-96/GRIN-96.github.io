# 모듈 패턴

- 모듈이란 ?
    
    외부에서 접근할 수 없는 데이터와 그 데이터를 제어하기 위한 함수로 구성된 구조물
    
- 장점
    - 데이터를 외부에서 직접 접근 불가
    - 사전에 정의된 함수로만 다룰 수 있다.
    - 함수의 내부구조나 데이터 구조를 마음대로 바꿀 수 있다.
    
- 큐
    - FIFO (First In First Out) 방식의 자료구조
    - 여러 값이 있을 때 제일 앞의 값이 처리되고 새로 입력된 값은 맨뒤에서 기다리는 구조이다.
    

📢 Enqueue : 줄의 맨 뒤에 데이터 추가

Dequeue : 줄의 맨 앞에 데이터 가져옴

Size : 줄의 길이
