# 벡터

- 벡터 - 데이터 접근
    - x[n] : 벡터 x의 n번째 요소
    - x[-n] : n번째 요소 제외한 나머지
    - x[start : end] : start 부터 end 까지의 값
    - length : 객체의 길이 반환
    - NROW : 배열의 행 또는 열의 수 반환

- 벡터 - 연산자
    - value %in% x : 벡터 x에 value가 저장되어 있는지 판단
    - x + n : 벡터 x의 모든 요소에 n의 값 더하기 (*, /, ^ 등 사용 가능)
